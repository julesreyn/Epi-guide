Function in c

Introduction​

In this tutorial, we are going to learn about functions in C. Functions are a very important part of C programming.
Functions are used to perform a specific task, we can also use them to split our program into multiple part. This is a very important part of the tutorial, so make sure you understand everything.

What is a function?​

A function is a block of code that performs a specific task. Print text, calculate a value, etc.
There are used to split our program into multiple parts and know where to start and where to end.
This give us a better overview of our program and make it easier to understand.

How to create a function?​

To create a function, we need to use a data type, a name and a set of parenthesis.

  • The data type is the type of data that the function will return. if we return nothing, we can use void.
  • The name is the name of the function.
  • The parenthesis are used to pass arguments to the function.

The function will be closed with a curly bracket.

data_type name(char arguments)
{
    // code
    return data_type;
}

How to call a function ?​

To call a function, we need to use the function name and pass the arguments.
The arguments are the data that we want to pass to the function. This is optional but useful.

void function(char arguments)
{
    write(arguments);
}

void main()
{
    char arguments = "Hello World";

    function(arguments);
}

How to return a value from a function ?​

To return a value from a function, we need to use the return keyword.
This can be used to return a value from a function if you want to use it later.

int add(int a, int b)
{
    return a + b;
}

void main()
{
    int a = 5;
    int b = 10;

    int result = add(a, b);
    write(result);

}

Output​

15

How to pass arguments to a function ?​

To pass arguments to a function, we need to use the function name and pass the arguments.
The arguments are the data that we want to pass to the function. This is optional but useful.

void function(char arguments)
{
    write(arguments);
}

void main()
{
    char arguments = "Hello World";

    function(arguments);
}

How to pass arguments by reference ?​

To pass arguments by reference, we need to use the & operator.
This will pass the address of the variable to the function.

Reminder :

  • The & operator is used to get the address of a variable. Where the data is stored.
  • The * operator is used to get the value of a variable. With the address of the variable.
void function(int *a)
{
    *a = 10;
}

void main()
{
    int a = 5;

    function(&a);
    write(a);
}

Output​

10

Conclusion​

In this tutorial, we learned about functions in C.
We learned how to create a function, how to call a function, how to return a value from a function, how to pass arguments to a function and how to pass arguments by reference.
This is a very important part of the tutorial, so make sure you understand everything. Have a nice coding session and see you in the next tutorial !
