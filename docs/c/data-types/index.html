Data Types

Introduction​

Data type may seem complicated at first, but it's actually quite simple. In this section, we'll go over the basics of data types and how to use them.

What is a data type ?​

A data type is a classification of data which tells the compiler or interpreter how the programmer intends to use the data.
As human beings, we can easily identify the type of data just by looking at it. For example, if we see the number 5, we know that it's an integer. If we see the number 5.0, we know that it's a floating point number. If we see the word "Hello", we know that it's a string.
But how does the computer know what type of data it is ?
The answer is that the programmer tells the computer what type of data it is.
We can do this by using data types.

Data Types​

There are 7 main data types in C :

  • Integer (int)
  • Floating Point (float)
  • Character (char)
  • Double (double)
  • Void (void)
  • Boolean (bool)

Lets go over each one of them.

Integer​

An integer is a whole number like "3" or "100".
In C, an integer can be either a short, an int, or a long.

Data TypeSizeRange
short2 bytes-32,768 to 32,767
int4 bytes-2,147,483,648 to 2,147,483,647
long8 bytes-9,223,372,036,854,775,808 to 9,223,372,036,854,775,807

Floating Point​

A floating point number is a number with a decimal point like "3.14" or "100.0".
In C, a floating point number can be either a float or a double.

Data TypeSizeRange
float4 bytes1.2E-38 to 3.4E+38
double8 bytes2.3E-308 to 1.7E+308

Character​

A character is a single letter or symbol like "a" or "!" or "👍".
In C, a character is represented by a single quote. 

char letter = 'a';
char symbol = '!';
char emoji = '👍';

Void​

Void is a special data type that is used to define functions that do not return a value.
In C, void is represented by the keyword void. 

void printHello()
{
    printf("Hello");
}

In some case, void can be a data type in function parameters and take the place of any other data type.
If your function can take a parameter of any type, you can use void.

Boolean​

A boolean is a data type that can only be true or false.
In C, a boolean is represented by the keyword bool.
It's very useful to make conditionnal statements. 

bool isTrue = true;
bool isFalse = false;
#include <stdio.h>
#include <stdbool.h>

int main() {
    bool isTrue = true;
    bool isFalse = false;

    if (isTrue) {
        write("isTrue is true\n");
    } else if (isFalse) {
        write("isFalse is true\n");
    } else {
        write("isTrue and isFalse are false\n");
    }
    return 0;
}
